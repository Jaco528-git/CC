# 每日政策时事简报（三轨制）

## 核心思路

每天生成**三份文稿**，分别放入三个文件夹：

| | A 轨：时效快报 | B 轨：政策专题深度 | C 轨：省情专题 |
|---|---|---|---|
| **文件夹** | `备考日报/A-时效快报/` | `备考日报/B-政策专题/` | `备考日报/C-省情专题/` |
| **目标** | 知道"今天发生了什么" | 理解"这个政策到底怎么回事" | 吃透"湖南省到底什么样" |
| **时间跨度** | 当日/当周 | 近 1-3 年 | 长期（固有省情） |
| **排序逻辑** | 按时间倒序 | 按重要性排序 | 按主题系统推进 |
| **内容密度** | 快讯式 + 热点深评 + 完整范文 | 深度单篇 + 完整范文 | 系统梳理 + 申论范文（真题导向） |
| **文件名** | `YYYY-MM-DD-快报.md` | `序号-主题名.md` | `序号-主题名.md` |

---

## 执行流程（必读 — 每次触发 skill 时优先执行）

### 强制规则（最高优先级，使用者指令不得覆盖）

1. **ABC 三轨缺一不可。** 无论使用者提到"双轨""只要A轨""先做B再做A"还是任何看似只要求部分轨道的表述，A、B、C 三条轨道都必须完整生成。使用者的表述仅表示其当前关注重点，不代表可以跳过其他轨道。
2. **始终并行，不得串行。** 三份文稿各自独立、互不依赖，必须在同一轮中同时启动三个子 Agent，不得等待上一轨完成再启动下一轨。使用者要求的顺序偏好（如"B更重要"）只影响审核阶段投入的精力配比，不影响启动方式。

### 操作步骤

1. **并行启动三个子 Agent**（同一轮同时发出，不等待）：
   - **Agent-A（时效快报）**：提示它**先读取昨日快报**（`备考日报/A-时效快报/YYYY-MM-DD-快报.md`）了解已覆盖内容，再按 Part A 搜索策略搜索当日新闻 + 范文材料 + 公考面试真题参考 → 按 Part A 格式撰写快报到 `备考日报/A-时效快报/YYYY-MM-DD-快报.md`，**避免与昨日内容重复**。提示中须告知本文件（SKILL.md）的路径，让子 Agent 能自行读取格式规范。
   - **Agent-B（政策专题）**：提示它读取本文件末尾的「附录一：B轨专题主题索引」，找到下一个未完成主题（无 ✅ 标记的第一条），按 Part B 结构搜索并撰写到 `备考日报/B-政策专题/序号-主题名.md`。任务完成后由主 Agent 在本文件的索引中标记 ✅ 和日期。
   - **Agent-C（省情专题）**：提示它读取本文件末尾的「附录二：C轨省情主题索引」，找到下一个未完成主题（无 ✅ 标记的第一条），按 Part C 结构搜索并撰写到 `备考日报/C-省情专题/序号-主题名.md`。任务完成后由主 Agent 在本文件的索引中标记 ✅ 和日期。
2. **审核检查**：三个 Agent 完成后，逐一检查文稿质量：
   - 来源链接是否齐全（每条信息必须带链接）
   - 时事热点是否按要求做了判断取舍和点评
   - 面试模拟题是否有具体限定条件，答题要点是否深入而非空话
   - 申论范文是否标注了作答要求
   - 格式是否符合本文件规范
   - A 轨是否与昨日快报去重（重复率应 < 20%）
3. **更新索引**：在本文件末尾的附录一/二中，将本次完成的主题标记为 ✅ 并填写完成日期。
4. **重建阅读器**：运行 `python 备考日报/build_reader.py`
5. **报告用户**：简要告知今日三份文稿的标题和位置。

### 注意事项

- B 轨和 C 轨是核心价值所在，合计投入 70% 以上精力
- C 轨申论范文优先使用湖南近年真题命题，体现"真题导向"
- 范文优先找现成的，找不到再自己写；自己写时必须严格遵循申论考试标准
- 所有政策表述必须使用官方原文，不可自行改写关键术语
- **每条信息必须附来源链接**，这是硬性要求
- 来源汇总放在每篇文稿末尾
- A 轨时事热点的面试模拟**不是每条都配**，先判断讨论度再决定
- A 轨面试模拟出题前，先搜索了解近年各地公考面试真题风格，确保题目有具体限定条件
- **所有索引状态以本文件正文中的附录一、附录二为准**，不再需要单独的索引文件

## 全平台通用规则

### 一、申论范文（ABC轨均必须包含）

每条轨道的文稿末尾必须包含一篇**完整的申论大作文范文**。

**范文获取优先级：**
1. **优先摘录现有优秀文章**：用 WebSearch 搜索当日/当主题相关的 人民日报评论、半月谈、新华社评论、学习时报 等发表的评论文章，选取最贴近主题的一篇，全文或精华段落摘录
2. **无合适文章时自行写作**：按照国考/省考申论大作文标准撰写

**搜索范文的专用关键词：**
- `"主题关键词" 人民日报评论 申论`
- `"主题关键词" 半月谈 评论`
- `"主题关键词" 人民时评 范文`
- `site:people.com.cn "主题关键词" 评论`

**范文要求：**
- 字数：1000-1200 字（国考标准）
- 结构：**申论题目（作答要求）** + 标题 + 开头（破题立论） + 3-4 个分论点（每段 200-250 字） + 结尾（升华收束）
- **必须在范文正文前标注申论题目**（即作答要求/给定主题），格式为 `> 作答要求：[题目]`，让读者能够判断范文如何切题、有无偏题
- 语言：规范、严谨、有理论高度，使用官方规范表述
- 内容：A 轨紧密结合当日热点；B 轨紧密结合专题主题；C 轨以近年湖南相关真题为范文题目
- 标注：在范文末尾注明是"摘录"还是"自撰"，摘录需注明原始出处

**范文在各轨中的位置：**
- A 轨：第六节，在"数据金句速记"之后
- B 轨：第六节，在"相关数据与案例"之后
- C 轨：第五节，在"考试视角"之后

### 二、配图规则

**对于适合用图片辅助说明的专题内容，应搜索并嵌入相关图片。**图片能显著提升阅读体验，尤其是地理、数据、文化等视觉性强的主题。

- **适用场景**：地理地图、数据图表、历史时间线、文化遗产照片、交通网络示意图等
- **不适用场景**：纯政策文本解读、时事新闻快讯（多数新闻不需要图，不强求）
- **图片格式**：使用 Markdown 图片语法 `![描述](图片URL)`
- **图片来源**：在 WebSearch 时顺带搜索图片相关关键词，或使用政府网站/百度百科等权威来源的配图
- **质量要求**：图片清晰、来源可靠、与内容紧密相关，宁缺毋滥

### 三、新闻来源链接（必须执行）

**每条新闻/信息/数据的末尾必须附上原始链接**，格式：
```
- 来源：[来源名称](URL)
```

对于通过 WebSearch 获取的信息，必须将搜索结果中的 URL 一并附上。不可只有来源名称而无链接。

**A 轨快报**：每条政策要闻、时事热点、省情快讯后必须带来源链接
**B 轨专题**：数据表格中的每条数据、每个案例后必须带来源链接
**C 轨省情**：每条数据、每个案例后必须带来源链接
**所有轨末尾**：各添加一个「## 来源汇总」章节，列出本篇所有引用的 URL

## Part A：时效快报

### 防重复规则（每日执行前必读）

**每个 A 轨快报必须是当日独有的增量内容，不得与前一日快报大面积重复。**

1. **写前先读**：撰写今日快报前，必须先用 Read 工具浏览昨日快报（`备考日报/A-时效快报/YYYY-MM-DD-快报.md`），了解哪些内容昨天已经覆盖，做到心中有数
2. **增量优先**：
   - 已在昨日快报中详细报道的政策/热点，今日**不再重复**核心内容和解读，仅在有实质性新进展时才简要跟进（标注"此前报道详见X月X日快报"）
   - 今日政策要闻优先选择**当日新发布/新公开**的文件和新闻
   - 同一事件跨多日发酵的，每日只写**当日的增量进展**，不把前情再复述一遍
3. **交叉检查**：成稿后与昨日快报对比，重复率应控制在 **20% 以内**（即 80% 以上内容应为本日独有）
4. **不可避免的重复**（如领导人连续多日的重要讲话、重大会议持续数日）：允许重复，但必须换角度——昨日从政策内容角度写，今日从社会反响/地方落实/考试命题角度写

### 搜索策略

并行搜索以下来源（8-10 路 WebSearch）：

1. `site:gov.cn 国务院 政策 当前月份` — 国务院最新政策
2. `人民日报 要闻 当前日期` — 党媒头条
3. `新华社 时事 当前日期` — 官方通讯社
4. `中国 经济数据 社会热点 当前月份` — 宏观数据
5. `湖南 时政 红网 当前月份` — 湖南本地时政
6. `site:hunan.gov.cn 湖南 政策 当前月份` — 湖南省政策
7. `公务员考试 申论热点 2026` — 备考风向
8. `2026 时政考点 国考 省考` — 考试相关
9. `公考面试真题 时事热点 湖南选调` — 面试真题参考（用于模拟题出题）
10. `"今日热点关键词" 人民日报评论 范文` — 为范文板块收集素材

解读点评的写法要领：

## 二、时事热点（2-3条）

**每条热点的处理流程：先判断 → 再决定配什么。**

判断标准：这个热点有没有公共讨论价值？是否反映深层矛盾/趋势变化/政策节点？如果是纯信息通报或常规事件，只写梗概+点评，不配面试模拟。只有具备思辨空间、可多角度分析的话题才配面试模拟。

**1. [热点]**
- 事件梗概：2-3句话概括
- 官方表态/各方反应：官方怎么说？社会舆论焦点在哪？
- 点评：为什么这件事值得关注？反映什么趋势或深层问题？放在更长的政策脉络或社会变迁中看，它在什么节点上？2-3句，要有观点而非复述事实。
- 面试模拟（**仅当话题有足够讨论深度时添加**）：
  > 「[完整题目 — 必须参考各地/各年份公考面试真题风格，题目要有具体限定条件，如"结合你的专业背景""结合基层工作实际""在AI技术快速发展的背景下""假如你是某乡镇干部"，拒绝泛泛而谈的出题方式]」
  >
  > 答题要点（**深入分析，拒绝空话套话**）：
  > ① 破题：点出问题的本质/矛盾核心（不是复述题目）
  > ② 分析：从2-3个具体角度展开（如制度层面、利益层面、执行层面），每个角度要有具体论据或案例支撑
  > ③ 对策：提出可落地的具体措施，至少一条要有操作细节（谁、做什么、怎么落实）
  > ④ 升华：结合国家治理现代化的宏观背景，但不喊口号
- 来源：[来源名称](URL)

**面试模拟出题参考来源：**
- 国考面试真题（各部委结构化面试）
- 湖南省考/选调面试真题
- 江苏/浙江/广东等省考面试真题（出题质量高，参考价值大）
- 注意：不是照搬真题，而是学习其出题角度和限定方式

**面试模拟答题深度要求：**
- 禁止出现：「加强组织领导」「完善体制机制」「加大投入力度」等无具体内容的空话
- 要求每一条要点都能落到具体行为或具体数据上
- 能用案例说明的用案例，能用数据支撑的用数据

## 四、行测常识快练（3道选择题）

### 出题核心原则

**行测常识不是完形填空，考的是理解而非背诵。** 题目可以与本日政策、时事内容强相关，但不能变成"从文章里抠掉一个词让你填"。每道题都应该让考生需要**思考推理**，而非单纯回忆原文。

### 题目类型配比

- 题1（单选）：概念理解型 — 考对政策/法律/理论的理解，选项需有辨析价值
- 题2（单选）：应用推理型 — 给场景或案例，要求判断适用哪个政策/原则/分类
- 题3（多选）：综合判断型 — 4个选项中2-3个正确，每个选项是一个独立知识点

### 出题质量标准（每道题逐一检查）

**1. 反"完形填空"检查**
- ❌ 禁止：直接从正文中抠掉数字/名词/文件名称让你填 — "《XX意见》提出到2030年服务业总规模达到多少万亿？"
- ❌ 禁止：问某会议/文件"强调的关键词有"然后让选原文表述组合 — 这是考记忆力不是考理解
- ✅ 应该：把知识点转化一个角度来问 — 不问你"目标是多少"，而问你"这个目标意味着什么"；不问你"文件叫什么"，而问你"这个文件解决什么问题"

**2. 辨析价值检查**
- 每个错误选项必须有迷惑性，但不是瞎编 — 要错得有道理（如：张冠李戴、时间错位、概念混淆、程度夸大/缩小）
- "17%"如果是正确答案，那么"15%"应该来自相近政策或常见口径，而非随机数字

**3. 考试风格对齐检查**
- 题目风格应对标：国考行测常识真题、省考行测常识真题、主流机构（粉笔/华图/中公）模拟题
- 特征：题干精炼（不超过3行）、选项对称（长度/格式一致）、考点有层次（不是表面信息）
- 多选題的每个选项应为独立判断点（不互为前提），正确选项间有逻辑关联（同属一个主题或政策体系）

**4. 内容关联检查**
- 题目考点应与本日快报的政策/时事内容有内在联系（共享同一个知识领域），但**不直接抄原文表述**
- 如果快报提到某政策设定了100万亿目标，常识题可以考"以下哪项属于'十五五'时期的量化约束性指标？"（包含该目标但也包含其他同类指标）

### 真题风格参考

以下为国考/省考行测常识的典型出题方式，出题时务必参考：

- **概念理解型**（单选）："根据《法治政府建设实施纲要》，以下关于重大行政决策程序的说法，正确的是？" → 考对制度的理解，4个选项各是一个可能的理解方向
- **应用推理型**（单选）："某市计划将一废弃矿区改造为光伏发电基地。根据'十五五'规划，该项目最可能获得以下哪项政策工具的支持？" → 给场景，判断政策工具
- **综合判断型**（多选）："关于碳排放双控，以下理解正确的有：①... ②... ③... ④..." → 每个选项是独立的知识判断

### 解析要求

每道题必须附带解析，解析要：
- 点明**正确选项为什么对**（政策依据/法律依据/逻辑推导）

- 可附带**记忆技巧**（口诀、联想、对比等）帮助考生举一反三
