# Claude Code 常用指令参考

## 一、斜杠命令（Slash Commands）

### 会话管理
| 命令 | 说明 |
|------|------|
| `/help` | 获取帮助信息 |
| `/clear` | 清除当前对话历史，重新开始 |
| `/resume` | 恢复之前的会话（可选择指定会话）|
| `/rewind` 或 `/undo` | 回退到上一个检查点（连续按 Esc Esc 可快速调出）|
| `/compact` | 压缩对话历史，释放 token 空间 |
| `/recap` | 返回会话时概括之前的上下文（v2.1.108+）|
| `/fork` | 分支当前会话，探索不同方向 |
| `/context` | 可视化查看 token 用量 |
| `/usage` | 统一查看费用 + 统计（替代 `/cost` + `/stats`）|

### 执行控制
| 命令 | 说明 |
|------|------|
| `/model` | 切换 AI 模型（如 opus / sonnet / haiku）|
| `/effort` | 控制推理深度：`low` / `medium` / `high` |
| `/fast` | 切换快速模式（Opus 加速输出）|
| `/config` | 查看/修改设置（主题、模型、权限等）|
| `/loop <间隔> <指令>` | 循环执行任务，如 `/loop 10m /daily` |
| `/sandbox` | 沙箱隔离模式 |
| `/powerup` | 交互式功能教程（v2.1.90+）|
| `/branch` | 会话分支（v2.1.69+）|

### 项目与文件
| 命令 | 说明 |
|------|------|
| `/init` | 自动生成 CLAUDE.md 项目说明 |
| `/memory` | 编辑项目记忆文件（CLAUDE.md）|

### 任务与后台
| 命令 | 说明 |
|------|------|
| `/tasks` | 管理后台 Bash/Agent 任务 |

### 其他
| 命令 | 说明 |
|------|------|
| `/plan` | 计划模式 — 先分析不出手 |
| `/goal` | 设定目标，让 AI 自主规划执行路径 |
| `/review` | 审查当前分支的 PR |

> 在输入框中输入 `/` 后按 **Tab** 可查看当前版本所有可用命令。

## 三、记忆系统（Memory）

我会记住你在对话中告诉我的重要信息，并在后续对话中调用。

- **持久化记忆**：存储在 `C:\Users\31070\.claude\projects\...\memory\` 目录
- **自动调用**：相关记忆会在对话中自动载入
- **手动记忆**：说"记住 xxx"我会立即保存，说"忘记 xxx"我会删除

记忆类型：用户信息（角色/偏好）、反馈（行为指导）、项目信息（目标/背景）、参考信息（外部资源位置）。

## 四、代理模式（Agent）

复杂任务可以派生子代理并行执行：

| 场景 | 说明 |
|------|------|
| 搜索代码 | `Explore` 代理 — 快速只读搜索 |
| 架构设计 | `Plan` 代理 — 先出方案再实现 |
| 研究调研 | `general-purpose` 代理 — 复杂多步任务 |

用法：说"帮我查一下 xxx"、"研究一下 xxx"等，我会自动启用代理。

## 五、常用功能速查

### 文件操作
- **读取文件**：`Read` — 支持代码、图片、PDF、Jupyter 笔记本
- **查找文件**：说"找到 xxx 文件" → Glob 搜索
- **搜索内容**：说"搜索 xxx" → Grep 全文搜索
- **编辑文件**：精确替换，只改需要改的部分
- **创建文件**：`Write` — 创建新文件

### 命令执行
- 直接说需要执行的命令即可
- **长时间任务**：说"在后台运行 xxx" → 后台执行，完成后通知
- **! 前缀**：需要我执行的 shell 命令（如 `! gcloud auth login`）

### Web 功能
- **网页搜索**：直接问"搜索 xxx" → 联网搜索（含来源链接）
- **网页抓取**：发链接给我 → 自动抓取并分析内容

### 任务管理
- 复杂任务我会自动创建任务列表并跟踪进度
- 可以用"进度"、"当前任务"等查看状态
