## 一、政策脉络

### 从"统一市场"到"全国统一大市场"：三十年演进

全国统一大市场并非新概念，而是中国市场化改革的核心命题之一。从十四大提出"统一市场"到二十届中央财经委专题部署，经历了从方向性表述到系统性制度建构的跃迁。

**关键节点时间线：**

| 时间 | 事件/文件 | 核心意义 |
|------|----------|---------|
| 1992年 | 十四大提出"形成统一的商品市场和要素市场" | 首次在国家战略层面提出"统一市场"方向 |
| 2013年 | 十八届三中全会提出"建设统一开放、竞争有序的市场体系" | 使市场在资源配置中起"决定性作用" |
| 2020年 | 十九届五中全会提出"加快构建以国内大循环为主体、国内国际双循环相互促进的新发展格局" | 统一大市场成为"双循环"战略的制度底座 |
| 2021年12月 | 中央深改委第二十三次会议审议相关文件 | "五统一"框架初步酝酿 |
| 2022年4月 | **《中共中央 国务院关于加快建设全国统一大市场的意见》正式印发** | 顶层设计文件出台——"立破并举"，拉开系统性建设帷幕 |
| 2022年10月 | 二十大提出"构建全国统一大市场，深化要素市场化改革" | 将统一大市场写入二十大报告，升格为党的意志 |
| 2023年12月 | 中央经济工作会议提出"加快全国统一大市场建设" | 在"加快"二字上持续加力 |
| 2024年7月 | 二十届三中全会部署"构建高水平社会主义市场经济体制" | 将统一大市场置于"高水平"框架中 |
| 2025年1月 | 国家发改委发布**《全国统一大市场建设指引（试行）》** | 从"要求做的、禁止做的、鼓励做的"三维度给出操作指南 |
| 2025年7月 | **习近平总书记主持二十届中央财经委第六次会议，专题研究全国统一大市场建设** | 最高层专题部署，讲话全文刊发于2025年第18期《求是》 |
| 2025年12月 | 中央经济工作会议提出"制定全国统一大市场建设条例" | 从政策文件迈向法治化 |
| 2026年3月 | "十五五"规划纲要专章部署，政府工作报告将"纵深推进"列为首要改革任务 | 进入"精耕细作"新阶段 |

### 领导人关键论述

> **习近平（2025年7月，二十届中央财经委第六次会议）**："构建全国统一大市场，是构建新发展格局的基础支撑和内在要求。要处理好'立'与'破'的关系，坚持立破并举、先立后破，在立的基础上破，在破的过程中立。"

> **习近平（2022年4月，关于《意见》的说明）**："建设全国统一大市场，不是要回到计划经济的老路上去，而是要更好发挥市场在资源配置中的决定性作用，更好发挥政府作用。"

## 三、核心概念解析

### 1. "五统一、一开放"

最新框架（2025年中央财经委第六次会议定型）：

| 维度 | 核心内涵 |
|------|---------|
| **统一市场制度规则** | 产权保护、市场准入、公平竞争、社会信用——四大基础制度全国统一 |
| **统一要素和资源市场** | 土地、劳动力、资本、技术、数据、能源——六大要素跨区域顺畅流动 |
| **统一商品和服务市场** | 产品质量标准、消费者权益保护规则全国一致 |
| **统一市场监管** | 监管规则、执法标准、信用惩戒"一把尺子量到底" |
| **统一市场设施联通** | 现代流通网络、市场信息交互渠道、交易平台互联互通 |
| **高水平开放** | 以国内统一大市场为"引力场"，集聚全球先进要素资源 |

"一破除"——破除地方保护和市场分割，是贯穿以上六个方面的主线。

### 2. "立破之道"（立破并举、先立后破）

| | "立"（建设统一规则） | "破"（拆除地方壁垒） |
|---|---|---|
| 制度层面 | 制定统一的市场准入负面清单（106项） | 清理废除地方保护性规定（已修订/废止2300余件文件） |
| 要素层面 | 建立全国统一的技术/数据交易市场 | 打破劳动力市场的户籍分割、土地市场的地方垄断 |
| 监管层面 | 推行"双随机、一公开"全国统一监管标准 | 整治"多头执法""重复检查"（涉企检查总量降33%） |
| 开放层面 | 以RCEP等高标准规则倒逼国内改革 | 消除内外贸"两张皮"现象 |

### 3. "内卷式"竞争

指地方政府为争夺企业/项目，在税收优惠、财政补贴、土地价格等方面展开恶性博弈，导致"低水平重复建设—产能过剩—压缩利润—继续内卷"的恶性循环。2026年政府工作报告明确"整治内卷式竞争"，拟出台**招商引资鼓励和禁止事项清单**，从源头遏制。

### 4. 统一大市场 vs 高标准市场体系

两者是"一体两面"：统一大市场是高标准市场体系的**空间维度**（打破地域分割），高标准市场体系是统一大市场的**质量维度**（提升制度标准）。简单说：统一大市场解决"通不通"的问题，高标准市场体系解决"好不好"的问题。

## 四、考试视角

### 申论价值

全国统一大市场在申论中可能出现的角色：

- **作为"背景"**：在分析地方保护主义、招商引资乱象等问题时，以统一大市场作为改革方向的宏观背景
- **作为"对策"**：对"打破市场壁垒""规范地方招商行为""降低物流成本"等议题提供制度性解决方案
- **作为"意义"**：论证建设统一大市场对构建新发展格局、释放超大规模市场红利的基础性作用
- **作为"关系分析"**：分析"政府与市场""中央与地方""内循环与外循环"等关系

### 面试真题模拟（3道）

**例题1 — 综合分析**

> 「当前一些地方政府为了招商引资，纷纷出台"超常规"优惠政策，如零地价供地、全额税收返还等。2026年政府工作报告明确提出"整治内卷式竞争"，将出台招商引资鼓励和禁止事项清单。请结合全国统一大市场建设，谈谈你对这一现象的看法。」

答题要点：
① **定性**：这不是"积极性高"，而是"制度性内耗"——地方竞相"放血"引来的不是优质企业，而是"税收候鸟"（哪里优惠去哪里的套利型企业），最终损害的是国家整体税收基础和公平竞争环境。
② **根源分析**：三个层面的原因——一是考核指挥棒偏差（GDP竞赛→招商竞赛），二是财政体制激励（留成比例博弈），三是缺乏全国统一的招商规则（底线不清、红线不硬）。
③ **对策**：制定"鼓励清单"（引导招什么——产业链补链强链型）和"禁止清单"（禁止做什么——不得违规减免税收、不得低于成本供地），同时将"统一大市场建设成效"纳入地方政府绩效考核。

**例题2 — 计划组织/应急处突**

> 「你是某省发改委的一名工作人员。近期国务院督查发现你省有三个地级市在招投标中存在设置"隐形门槛"的问题——要求投标企业"在本地注册满三年""有本地纳税记录"等。省长要求你起草一份整改方案，并在一个月内向省政府常务会汇报落实情况。请谈谈你的工作思路。」

答题要点：
① **摸底定底数**：一周内完成全省14个地级市招投标文件专项审查，对照《全国统一大市场建设指引》"禁止做的"15条清单逐条排查，形成"问题清单"和"整改台账"（注明每条违规条款的政策依据和整改时限）。
② **整改分两步**：第一步"废"——两周内废止所有被点名三市的违规条款，同步全省各地自查自纠；第二步"立"——制定《全省招投标负面清单管理办法》，"一把尺子量到底"，各地不得自行增设准入条件。
③ **长效靠制度**：建立"双审查"机制——新出台的招投标文件在发布前，同时接受"合法性审查"（法制办）+ "统一市场性审查"（发改委/市场监管），从源头堵住"换马甲"式的隐性壁垒。

**例题3 — 观点思辨**

> 「有人说："建设全国统一大市场就是要让政府的手'收回来'，完全交给市场"；也有人说："统一大市场恰恰需要政府更有为，因为打破壁垒、制定规则都要靠政府推动。"请谈谈你的理解。」

答题要点：
① **两种观点各执一端，都有片面性**：前者误读了"统一大市场＝市场化"——忘记了统一规则本身就是"有为政府"的产出；后者忽视了统一大市场的终极目标是"让市场在更大范围内发挥决定性作用"。
② **正确定位：有效市场+有为政府**——习近平明确指出"不是要回到计划经济的老路上去"。统一大市场的核心命题不是"政府vs市场"，而是"好政府+好市场"。好政府制定统一的交通规则（制度规则），好市场让车辆（企业）在全国高速公路上自由通行。
③ **结合实践**：全国累计清理文件3.8万件、修订/废止2300余件——这些"破壁垒"的动作本身就是政府"有为"的体现。但"有为"不是"乱为"：招商引资禁止清单就是在给地方政府的"有为"划定边界——在边界内积极作为，在边界外坚决收手。

### 行测常识价值（附3道模拟题）

**模拟题1（单选·概念理解）** 关于全国统一大市场建设中的"五统一"框架，以下说法正确的是：

A. "五统一"包括统一市场制度规则、统一要素和资源市场、统一商品和服务市场、统一市场监管、统一财税体制
B. "统一市场监管"意味着全国所有市场监管事项均由中央垂直管理，地方不再保留市场监管部门
C. "统一要素和资源市场"涵盖土地、劳动力、资本、技术、数据和能源六大要素的跨区域流动
D. "五统一"是在2022年《关于加快建设全国统一大市场的意见》中首次完整提出的

> 答案：C。解析：C项准确描述了"统一要素和资源市场"的六大要素范畴。A项错误——"统一财税体制"不在"五统一"中，"五统一"的第五项是"统一市场设施联通"，财税制度改革是配套措施而非"五统一"本身。B项错误——"统一市场监管"指监管规则和执法标准全国统一，并非管理主体统一为中央；地方政府仍承担属地监管职责，只是不能"自立标准"。D项错误——"五统一、一开放"的完整框架是在2025年7月二十届中央财经委第六次会议上定型的，2022年《意见》奠定了方向但未形成"五统一"的规范表述。

**模拟题2（单选·应用推理）** A省某市住建局发布招标公告，要求投标企业在"本市注册经营满5年且近3年在本市有连续纳税记录"。根据全国统一大市场建设相关规定，该条款最可能被认定为：

A. 合法合理——有助于确保中标企业有本地服务能力和信誉基础
B. 属于"鼓励做的"——符合地方政府扶持本地企业的政策方向
C. 属于"禁止做的"——构成以注册地和纳税地限制市场准入的隐性壁垒
D. 属于"要求做的"——但设置"5年"时限过长，应缩短至3年

> 答案：C。解析：根据《全国统一大市场建设指引（试行）》"禁止做的"清单，不得违法设定与招标采购项目具体特点和实际需要不相适应的资质资格、技术、商务条件，不得以注册地、所有制形式、所在地等不合理条件限制经营者参与。以"本市注册经营满5年"和"本地纳税记录"为投标门槛，属于典型的"以注册地限制市场准入"的隐性壁垒。A项以"服务能力"为名行保护主义之实——企业在A市注册时间长短≠服务能力高低。B项完全错误——保护主义不属于"鼓励"。D项试图用"缩短年限"来"修正"一个本质错误的条款——问题不在年限长短，而在于"以注册地为门槛"本身就违规。

**模拟题3（多选·综合判断）** 2022年至2026年，全国统一大市场建设经历了从"立柱架梁"到"精耕细作"的升级。以下关于这一演进的理解，正确的有：（多选）

A. 2022年《意见》的出台标志着全国统一大市场从地方自发探索上升为国家顶层设计
B. 2025年《全国统一大市场建设指引（试行）》的发布意味着统一大市场建设从"政策宣示"进入"操作指南"阶段
C. 2026年政府工作报告提出"制定全国统一大市场建设条例"，表明建设重点从行政推动转向法治保障
D. 市场经济条件下，地方间的税收优惠竞争属于正常的市场行为，不应纳入统一大市场建设的约束范围

> 答案：BC。解析：B项正确——2025年《指引》从"要求做的、禁止做的、鼓励做的"三个维度给出具体操作清单，确实是从宏观方向到微观操作的跃迁。C项正确——《条例》的提法意味着统一大市场将从"政策驱动"升级为"法治驱动"，为市场主体提供更稳定的制度预期。A项错误——2022年之前统一大市场并非"地方自发探索"，而是从1992年十四大到2021年中央深改委，经历了长期的政策酝酿，2022年《意见》是国家层面的系统性顶层设计，而非"从地方到中央"的升级路径。D项错误——地方政府间的税收优惠竞争如果演变为"逐底竞争"（竞相减免税收拉企业），就构成了"内卷式竞争"，恰恰属于统一大市场建设要整治的对象；正常的市场竞争≠恶性的政策竞争。

## 六、申论范文

> 作答要求：围绕"有为政府与有效市场相结合，纵深推进全国统一大市场建设"这一主题，自拟题目，写一篇1000-1200字的议论文。

### 以"有为"政府护航"有效"市场

市场如水，天然具有流动和扩展的本能；壁垒如堤，人为割裂着经济循环的脉络。建设全国统一大市场，本质上是一场"拆堤通水"的深刻变革——让14亿多人口的超大规模市场真正成为一个整体，让生产要素在960万平方公里的土地上自由奔涌。这不是要回到计划经济的老路，而是要在更高水平上实现"有效市场"与"有为政府"的辩证统一。

**统一规则是市场"通"的前提。** 市场经济不是不要规则，而是要统一规则。过去一段时期，一些地方为保护本地企业，在招投标中设"注册地门槛"、在政府采购中列"本地品牌清单"、在监管执法中搞"选择性执法"——看似在"呵护"本地经济，实则是在用行政藩篱切割统一市场。2025年出台的《全国统一大市场建设指引（试行）》以"要求做的、禁止做的、鼓励做的"三个维度划出清晰红线，让地方政府知道"什么该做、什么不该做、什么值得做"。全国累计清理文件3.8万件、修订废止2300余件——这些数字背后，是同一个逻辑：统一规则不是束缚地方手脚的绳索，而是让所有市场主体"站在同一起跑线"的保障。

**深化要素市场化改革是市场"活"的关键。** 商品市场的统一相对容易——物流网络四通八达，快递包裹畅通无阻。但要素市场的统一才是"硬骨头"：土地市场的地方垄断、劳动力市场的户籍分割、数据市场的"信息孤岛"、技术市场的"成果不出省"……这些深层次的制度性分割，才是制约经济循环的真正堵点。2026年政府工作报告提出"完善统计、财税、考核制度"，正是要解决要素流动背后的"制度激励"问题——当地方政府的GDP核算不再"按注册地算"而是"按实际贡献算"，当财税分配不再鼓励"税收洼地"恶性竞争，要素自然会流向效率最高的地方。

**整治"内卷式"竞争是市场"好"的底线。** "零地价供地""全额税收返还""电价一毛钱补贴"——一些地方政府为了抢项目、抢企业，竞相"放血"招商。看似"拼服务"，实则"拼谁更不计成本"。结果是：企业成了"税收候鸟"——哪儿补贴高飞哪儿，留给地方的只有日益沉重的财政负担和大量低效重复产能。2026年即将出台的"招商引资鼓励和禁止事项清单"，正是要给地方政府的招商行为划出"底线"：鼓励产业链补链强链型招商，禁止违法违规减免税收、低于成本供地。说到底，"好"的市场竞争不是拼谁能给企业"开小灶"，而是拼谁能给所有企业"做好饭"。

**推动制度型开放是市场"强"的引擎。** 全国统一大市场不是"封闭的国内市场"，而是以统一大市场为"引力场"，集聚全球先进要素资源。对53个非洲建交国全面实施零关税、积极对接CPTPP和DEPA高标准经贸规则、在自贸试验区率先试行"边境后"规则改革——这些举措的深层逻辑是：用国内统一大市场的规模优势，重塑中国在全球经济格局中的竞争新优势。一个14亿多人口的统一市场，本身就是对全球企业和人才最有吸引力的"邀请函"。

古人云："通则不痛，痛则不通。"中国经济体量已超130万亿元，如果内部还存在形形色色的"断头路"和"篱笆墙"，超大规模市场优势就难以充分释放。纵深推进全国统一大市场建设，做的是"疏经络"的功夫，求的是"高质量"的实效。让市场在更大范围内畅通无阻，让政府"有为"且"有界"——这既是构建新发展格局的基础支撑，也是中国式现代化的题中应有之义。
