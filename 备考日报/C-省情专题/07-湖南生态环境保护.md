## 一、概况

湖南省地处长江中游、洞庭湖以南，国土面积21.18万平方公里，是国家"一江一湖三山四水"生态安全格局的核心承载区。

**核心数据速览：**

- **水质**：国考断面I-III类水质优良率98.6%，居中部六省第一。长江干流湖南段和湘资沅澧四水干流连续五年达到或优于II类水质。劣V类水体比例为0%。
- **空气**：2024年空气质量优良率89%，优于全国平均水平1.8个百分点；PM2.5平均浓度36.1微克/立方米。
- **森林**：森林覆盖率59.98%，森林蓄积量6.97亿立方米，湿地保护率70.54%。
- **绿色转型**：近五年单位GDP能耗下降17.2%，非化石能源消费占比达24%，国家级绿色工厂271家，绿色园区27家。
- **生物多样性**：洞庭湖越冬候鸟47.41万只（2025-2026年），江豚194头，麋鹿近400头——"候鸟的欢歌、江豚的微笑、麋鹿的倩影"已成为洞庭湖三大生态名片。

## 二、历史沿革与政策脉络

湖南省生态环境保护的政策演进，历经"局部治污→流域治理→系统联治→绿色转型"四个阶段，呈现出一条清晰的政策升级路径。

| 时间 | 事件/文件/会议 | 意义 |
|------|--------------|------|
| 2013年 | 湖南省将湘江保护和治理列为全省"一号重点工程"，出台中国第一部江河流域保护综合性地方法规《湖南省湘江保护条例》 | 开启流域系统治理先河，以"堵源头"为目标启动第一个"三年行动计划" |
| 2013-2015年 | 第一个"三年行动计划"（堵源头）：关停退出1100多家涉重金属污染企业，五大重点区域集中整治 | 基本堵住湘江干流污染源头，清水塘261家企业全面关停退出（2018年完成） |
| 2016-2018年 | 第二个"三年行动计划"（治调并举）：主战场从湘江干流向"一湖四水"延伸，2273家规模畜禽养殖场退养，十大重点行业清洁化改造 | 2018年湘江干支流I-III类水质断面优良率达98.7%，较2012年提高10.6个百分点 |
| 2018年4月 | 习近平总书记考察岳阳，提出"守护好一江碧水" | 成为湖南生态保护总遵循，岳阳华龙码头变身"江豚湾"4A级景区 |
| 2019-2021年 | 第三个"三年行动计划"（巩固提高）：10方面30项具体任务基本完成 | 三个三年计划收官，长江干流湖南段和四水干流全部达到或优于II类 |
| 2020年1月 | 长江十年禁渔启动，湖南退捕渔船20376艘、渔民28588人全部退出，转产就业率100% | 长江流域生态休养生息的标志性举措 |
| 2021年 | 《湖南省洞庭湖保护条例》施行，"洞庭清波"专项行动常态化 | 洞庭湖保护进入法治化轨道 |
| 2024年5月 | 《关于全面推进美丽湖南建设的实施意见》发布 | 明确实施"一江一湖四水"生态修复工程 |
| 2024年9月 | 《湖南省空气质量持续改善行动计划实施方案》 | 到2025年PM2.5浓度力争控制在32微克/立方米以内 |
| 2024年11月 | 省人大常委会通过《湖南省水污染防治条例》（2025年5月施行） | 补齐水污染防治综合性法规短板，共38条 |
| 2024年12月 | 《湖南省加快经济社会发展全面绿色转型实施方案》 | 到2030年非化石能源消费比重达25% |
| 2025年5月 | 《关于加强生态环境分区管控的实施意见》发布 | 确立"一江一湖三山四水"生态安全格局，建立"1+4+14+N"管控体系 |
| 2025年9月 | 第11号省总河长令发布 | 构建"一核、两区、三屏、四廊"幸福河湖骨架 |

## 四、考试视角

### 申论价值

本专题在湖南省考/选调申论中出现概率极高，可作为以下方向的核心素材：

1. **"守护好一江碧水"主题**：习近平总书记2018年岳阳考察时的殷殷嘱托，是湖南生态环境保护的最高政治遵循。可用于论述生态文明建设的地方实践。
2. **洞庭湖生态治理**：总磷攻坚、十年禁渔、江豚麋鹿回归等，是"人与自然和谐共生"的鲜活案例。可用于论证"绿水青山就是金山银山"的转化路径。
3. **湘江保护与治理"一号重点工程"**：三个"三年行动计划"的滚动推进，体现"久久为功、系统治理"的方法论，适合作为对策型文章的政策工具参考。
4. **产业绿色转型**：清水塘"壮士断腕"关停261家企业、三十六湾橡树修复"治污+产业"双赢，可作为"高质量发展与高水平保护协同推进"的论据。
5. **生态环境分区管控**：2025年新出台的分区管控实施意见，是"精准治污、科学治污、依法治污"的制度样本。

**常见出题角度：** "以'守护好一江碧水'为主题，自选角度，写一篇文章"；"结合洞庭湖生态治理实践，谈谈对'绿水青山就是金山银山'的理解"。

### 面试真题/模拟（3道）

**例题1 — 综合分析（参考湖南选调面试风格）**

> 习近平总书记2018年在岳阳考察时嘱托湖南"守护好一江碧水"。近年来，洞庭湖江豚从110头增至194头，麋鹿从100多头增至近400头，越冬候鸟达47万余只。但也有群众反映："江豚多了是好事，但我们老百姓的收入怎么保障？"假如你是洞庭湖区某乡镇干部，请谈谈你的看法。

**答题要点：**

1. **破题定调**：这本质上是"绿水青山"与"金山银山"如何在基层实现统一的问题，不能将生态保护与民生改善对立起来。
2. **辩证分析**：生态好转本身可创造经济价值——君山区"江豚湾"年接待游客超50万人次；沅江"六门闸风干鱼"成为国家地理标志产品；芦苇转型芦笋/芦菇/芦花板产业。
3. **具体对策**：发展生态旅游，打造观鸟经济、生态研学品牌；推动绿色产业如稻虾共作、新能源船舶制造（产值132.9亿元）；落实退捕渔民社保全覆盖和转产就业帮扶。
4. **升华**：生态环境保护和经济发展不是选择题，而是辩证法。基层干部要学会在"保护"中寻找"发展"的空间，让老百姓端上"生态饭碗"。

- 参考来源：[中公教育 — 转型发展守护大美洞庭](http://www.offcn.com/mianshi/2022/0729/81784.html)

**例题2 — 计划组织**

> 你所在县位于湘江沿岸，近年来水质监测数据显示部分支流断面总磷浓度时有超标。县政府决定开展为期半年的入河排污口"查、测、溯、治"专项整治行动，由你负责制定实施方案。请问你会如何组织？

**答题要点：**

1. **摸清底数**：联合生态环境、水利、农业农村等部门，对县域内所有入河排污口全覆盖排查，建立"一口一策"台账（参照湖南省全域排污口排查模式）。
2. **分类施策**：工业排污口限期接入园区污水处理厂或提标改造；农业面源推进测土配方施肥（覆盖率已超95%）、畜禽养殖废弃物资源化利用；生活污水加快乡镇污水处理厂管网建设。
3. **科技赋能**：依托"智慧渔政"等数字化平台，布设水质在线监测微站，参考岳阳"电子河长"AI识别模式。
4. **压实责任**：将整治任务纳入河湖长制考核，参照君山区372个排污口排查、17个问题排口全部整治销号的经验。
5. **长效机制**：整治完成后定期"回头看"，推动生态环境分区管控制度落地。

A. "一江一湖三山四水"仅涵盖湖南省内的水域，不涉及省际协同治理
B. 洞庭湖区域明确规定禁止布局对生态系统有严重影响的项目，遏制重污染企业向湖区转移

### 以绿水青山作答，书写美丽湖南的时代答卷

湘江北去，洞庭浩渺。湖南因水而名、因水而兴、因水而美。2018年4月，习近平总书记在岳阳考察时深情嘱托"守护好一江碧水"，为湖南生态文明建设指明了方向。八年来，湖南人民牢记嘱托，以"一江一湖三山四水"为主战场，打响了一场波澜壮阔的生态保卫战。从湘江重金属污染治理的"壮士断腕"，到洞庭湖总磷攻坚的"久久为功"，三湘大地正在用行动证明：绿水青山就是金山银山，高水平保护与高质量发展可以同频共振。

**守护好一江碧水，必须坚持系统观念，统筹山水林田湖草沙一体化治理。** 湖南的生态环境问题，表象在"水"，根子在"岸"，核心在"人"。湘江曾是全国重金属污染最严重的河流之一——株洲清水塘、湘潭竹埠港、衡阳水口山、郴州三十六湾、娄底锡矿山五大片区每年向湘江排放大量工业废水。"头痛医头、脚痛医脚"显然行不通。湖南省以系统思维破题：2013年将湘江保护与治理列为全省"一号重点工程"，滚动实施三个"三年行动计划"，关停退出1182家涉重金属污染企业，治理水土流失1207平方公里，复绿矿山1500多公顷。更关键的是，湖南将治理范围从湘江干流逐步扩展至"一湖四水"全域，2025年又出台《关于加强生态环境分区管控的实施意见》，确立"一江一湖三山四水"生态安全格局，划定优先保护、重点管控和一般管控三类单元——这是从"末端治污"到"源头管控"的治理范式跃迁。系统治理之下，全省国考断面水质优良率达到98.6%，居中部六省第一；湘资沅澧四水干流连续五年保持II类水质。一条江的改变，折射出的是一套方法论的成功。

**守护好一江碧水，必须坚持精准治污，集中力量攻克突出生态环境问题。** 洞庭湖总磷超标是困扰湖南多年的"老大难"，也是长江经济带生态环境警示片点名的问题。总磷来源复杂——农业面源、城镇生活污水、工业排放、内源释放，牵一发动全身。湖南没有用"大水漫灌"式的粗放治理，而是精准锁定"病灶"：2022年出台《洞庭湖总磷污染控制与削减攻坚行动实施方案》，将任务细化到县、到断面、到污染源。岳阳排查畜禽养殖场户5769个，改造水产养殖池塘20772亩；君山区排查入河排污口372个，问题排口17个全部整治销号。到2025年，东洞庭湖断面总磷浓度降至0.06mg/L，首次稳定达到国家考核目标。洞庭湖总磷较2015年下降52.7%，一条"江豚的微笑曲线"在数据中浮现——江豚从110头增至194头，麋鹿从100多头增至近400头，越冬候鸟达47万余只。精准治污，让"八百里洞庭"重现碧波万顷。

**守护好一江碧水，必须坚持绿色转型，在保护中寻求发展新空间。** 有人担心，生态保护会阻碍经济发展——株洲清水塘关停261家企业，郴州三十六湾关停198家矿山，岳阳沿江化工企业"关搬转"35家，这些"减法"难道不是经济增长的代价？然而事实证明，关停不是终点，而是新生的起点。清水塘腾退后，轨道交通装备、新能源汽车等绿色产业加速聚集；三十六湾利用橡树修复重金属污染土壤，成材后亩产值可达3-5万元，实现"治污+产业"双赢；沅江市退捕渔民转型发展"六门闸风干鱼"产业，年产值过亿元，带动数千名渔民端上"生态饭碗"。湖南省发改委数据显示：近五年全省单位GDP能耗下降17.2%，非化石能源消费占比达24%，国家级绿色工厂达271家——绿色正在成为湖南发展的鲜明底色。

"人不负青山，青山定不负人。"今天的三湘大地，湘江水清岸绿，洞庭湖候鸟翔集，矿山废墟变身生态公园，化工老区长出新兴产业。守护好一江碧水，不仅是一场生态保卫战，更是一场发展观的深刻变革。面向"十五五"，湖南正以生态环境分区管控为抓手，加快经济社会发展全面绿色转型，在青山绿水间走出一条人与自然和谐共生的现代化之路。这是对习近平总书记殷殷嘱托的最好回答，也是对子孙后代最负责任的时代答卷。
