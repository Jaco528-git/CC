## 二、本届（二十届）各会议时间轴

```
2022年 ──── 10月 二十大（15-22日）
        ──── 10月 二十届一中全会（23日）
        ──── 12月 中央经济工作会议（15-16日）

2023年 ──── 1月 湖南省两会
        ──── 2月 二十届二中全会（26-28日）
        ──── 3月 全国两会（政府工作报告GDP目标5%左右）
        ──── 4月/7月/12月 季度政治局会议

2024年 ──── 1月 湖南省两会
        ──── 3月 全国两会（"人工智能+"首次写入）
        ──── 7月 二十届三中全会（15-18日，改革决定）
        ──── 9月 政治局会议（破例部署经济，最大政策转折点）
        ──── 12月 中央经济工作会议（"适度宽松"14年首次）

2025年 ──── 1月 湖南省两会
        ──── 3月 全国两会（十五五规划启动）
        ──── 10月 二十届四中全会（审议十五五规划建议）★本届特殊性
        ──── 12月 中央经济工作会议

2026年 ──── 1月 湖南省两会
        ──── 3月 全国两会（十五五规划纲要通过，GDP目标4.5%-5%区间）
        ──── 4月 政治局会议（"六网"建设首次提出）
        ──── 二十届五中全会（尚未召开，截至5月10日）
```

### ★ 本届特殊变化

传统上五中全会审议"五年规划建议"，但**本届改为四中全会（2025年10月）**承担此职能。这意味着五中全会（预计2026年下半年）的议题大概率聚焦**党建、意识形态或迎接二十一大**。这一调整是改革开放以来首次，值得在面试中作为"制度弹性"的论据。

## 三、会议间的内容承接关系

### 3.1 二十大报告 → 历次中央全会

| 二十大报告中的重点 | 承接的全会 | 具体体现 |
|-------------------|-----------|---------|
| "深化党和国家机构改革" | 二中全会（2023.2） | 审议《党和国家机构改革方案》，金融/科技/数据三大主线 |
| "坚持深化改革开放" | 三中全会（2024.7） | 审议改革决定，300+项改革举措，7大聚焦 |
| "到2035年基本实现社会主义现代化" | 四中全会（2025.10） | 审议十五五规划建议，2026-2030五年锚定 |
| "推进国家安全体系和能力现代化" | *预期五中全会* | 可能进一步部署 |

### 3.2 中央经济工作会议（12月）→ 全国两会（次年3月）

| 经济工作会议定调 | 两会政府工作报告量化 |
|-----------------|-------------------|
| 2022.12："稳字当头、稳中求进" | 2023.3：GDP目标5%左右，赤字率3% |
| 2023.12："稳中求进、以进促稳、先立后破" · 新质生产力 | 2024.3："人工智能+"写入，专项债3.9万亿 |
| 2024.12：货币政策"适度宽松"（14年首次）、"更加积极"财政 | 2025.3：赤字率4%，超长期特别国债1.3万亿 |
| 2025.12："稳中求进、提质增效" · 十五五开局 | 2026.3：GDP区间目标4.5%-5%，赤字率4% |

### 3.3 全国两会 → 湖南省两会（政策传导链：国标→省标）

实例——零基预算改革传导全过程：

```
中央层面：
  二十届三中全会（2024.7）→ 提出"深化零基预算改革"
  中央经济工作会议（2024.12）→ 部署"推进财政科学管理"
  全国两会（2025.3）→ 政府工作报告写入"开展零基预算改革试点"

湖南层面：
  省委经济工作会议（2024.12）→ 将"深化零基预算改革"列为2025年重点
  湖南省两会（2025.1）→ 政府工作报告明确"全面实施零基预算"
  湖南省两会（2026.1）→ 报告成效"省级专项从42项整合为五大类"
```

传导时间差：中央部署→湖南落实通常在3-6个月内完成，"三高四新"等长期战略则持续数年。

### 3.4 政治局会议对经济工作会议的"预热"与"纠偏"

| 时间 | 政治局会议信号 | 后续经济工作会议回应 |
|------|-------------|------------------|
| 2024.9.26 | **破例**在9月讨论经济，"加力推出增量政策" | 2024.12：货币政策14年首次"适度宽松" |
| 2025.7 | 强调"反内卷"、整治"招商引资恶性竞争" | 2025.12："提质增效"，强化统一大市场 |
| 2025.12 | 定调"宏观政策更加积极有为" | 为2025.12经济工作会议预热 |

### 4.2 行测常识"必背会议链"

以下知识点在国考/省考行测常识中频繁出现：

1. **全会职能口诀**：一中选领导、二中议人事、三中推改革、四中（本届）定规划、五中待定
2. **会议顺序**：党代会（10月）→ 一中全会（10月）→ 二中全会（次年2月）→ 两会（次年3月）
3. **经济政策传导**：中央经济工作会议（12月定调）→ 两会政府工作报告（3月量化）→ 政治局季度会议（过程监控）
4. **"首次"必考**：新质生产力（2023年经济工作会议首次）、"人工智能+"（2024年两会首次）、"适度宽松"（2024年经济工作会议14年首次）
5. **2026年特色考点**：GDP首次采用区间目标（4.5%-5%）、六网建设新提法
