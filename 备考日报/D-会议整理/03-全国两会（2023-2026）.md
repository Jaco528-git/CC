## 一、两会制度概述

### 1.1 "两会"是什么

"两会"是对**全国人民代表大会**（简称"人大"）和**中国人民政治协商会议**（简称"政协"）全国委员会会议的统称。两者每年3月在北京同期召开，是国家政治生活中的头等大事。

"两会"机制可追溯至1959年。周恩来总理在当年全国人大常委会与全国政协常委会联席会议上首次使用"两会"说法，指出：**"两个大会联合起来开会是一个新形式……人大、政协'两会'有不同之处，权力上有分别，但应该说'两会'只有权力之分，无高低之别。"** 从1985年起，两会固定在每年3月召开。自1998年起，全国政协会议一般在3月3日开幕、全国人大会议在3月5日开幕。

### 1.2 人大 vs 政协：职能区别

| 对比维度 | 全国人民代表大会（人大） | 中国人民政治协商会议（政协） |
|---------|--------------------------|-------------------------------|
| **性质** | 最高国家权力机关 | 中国人民爱国统一战线组织；多党合作和政治协商的重要机构；社会主义协商民主的重要渠道和专门协商机构 |
| **常设机关** | 全国人民代表大会常务委员会 | 全国政协常务委员会 |
| **产生方式** | 人大代表由**选举产生**（直接选举+间接选举） | 政协委员由**协商推荐产生** |
| **成员称谓** | 全国人大代表 | 全国政协委员 |
| **职权/职能** | 修改宪法、监督宪法实施；制定和修改法律；选举和罢免国家领导人；审查批准预算和计划；决定国家重大事项 | 政治协商、民主监督、参政议政 |
| **审议方式** | 人大"**审议**"报告，表决作出**决议**，有法律效力 | 政协"**讨论**"报告，提出意见和**建议**，不表决、不作决议 |
| **提交文书** | 人大代表提"**议案**"和"**建议**" | 政协委员提"**提案**" |
| **决策属性** | 权力机关，能决策、能立法 | 非权力机关，不决策、不立法，类似"大智库" |
| **十六字方针** | —— | 长期共存、互相监督、肝胆相照、荣辱与共 |
| **界别设置** | 按行政区域和选举单位 | 设**34个界别**（涵盖各党派、各团体、各民族、各界代表人士） |

**关键结论**：中国的两会**不是**西方的上议院和下议院。政协不构成国家权力机关的另一极，人大与政协之间是**相互补充、协调统一**的关系——选举民主与协商民主相结合，避免了西方议会中常见的政党掣肘、议而不决等问题。

### 1.3 两会与党代会/中央全会的关系定位

三者构成一条**"党定方向→全会作部署→两会法定化"**的递进式制度链条：

```
党代会（5年一次，顶层定向）
    ↓
中央全会（每年至少1次，战略分解）
    ↓
两会（每年3月，转化为国家意志）
```

| 会议 | 全称 | 周期 | 核心职能 |
|------|------|------|----------|
| 党代会 | 中国共产党全国代表大会 | 每5年一次 | 党的最高权力机关，确定重大方向与战略 |
| 中央全会 | 中央委员会全体会议 | 每年至少1次，每届约7次 | 落实党代会决策，议定各领域发展战略 |
| 两会 | 全国人大 + 全国政协 | 每年3月 | 将党的主张通过法定程序上升为国家意志 |

**历次中央全会的常规分工**：
- **一中全会**：选举产生党的领导机构
- **二中全会**：为"两会"准备国家机构人选推荐方案
- **三中全会**：通常聚焦经济与改革（如十八届三中全会"全面深化改革"，二十届三中全会聚焦"进一步全面深化改革、推进中国式现代化"）
- **四中至六中全会**：就依法治国、社会治理、规划建议等专题部署
- **七中全会**：为下一届党代会做准备

**制度衔接的经典路径**：党中央通过全会提出"五年规划建议" → 国务院编制规划纲要草案 → 次年两会由全国人大审查批准 → 将**党的主张转化为国家意志**。

**核心表述（申论金句）**：
- "党的主张通过法定程序转化为国家意志和人民共同行动"
- "把顶层设计和问计于民统一起来"
- "全过程人民民主的生动实践"

## 二、2023年全国两会

> 2023年3月5日，国务院总理李克强在第十四届全国人民代表大会第一次会议上作《政府工作报告》。
> 这是换届之年的政府工作报告，也是十四届全国人大一次会议的"开局"报告。

### 2.1 政府工作报告核心数据

| 指标 | 2023年目标 | 2023年实际 |
|------|-----------|-----------|
| GDP增速 | 5%左右 | **5.2%**（初核126.06万亿元；终核上调至129.43万亿元） |
| CPI涨幅 | 3%左右 | **0.2%**（远低于目标，内需偏弱） |
| 城镇新增就业 | 1200万人左右 | **1244万人** |
| 城镇调查失业率 | 5.5%左右 | 平均**5.2%** |
| 居民收入增长 | 与经济增长基本同步 | 实际增长**6.1%** |
| 粮食产量 | 1.3万亿斤以上 | **1.39万亿斤** |
| 赤字率 | 3% | — |
| 地方政府专项债券 | 3.8万亿元 | — |
| 单位GDP能耗 | 降低2%左右 | — |

### 2.2 重点政策部署

1. **着力扩大国内需求**：把恢复和扩大消费摆在优先位置，稳定大宗消费，推动生活服务消费恢复。
2. **加快建设现代化产业体系**：集中优质资源推进关键核心技术攻关，支持平台经济发展。
3. **切实落实"两个毫不动摇"**：依法保护民营企业产权和企业家权益。
4. **更大力度吸引和利用外资**：扩大市场准入，积极推动加入CPTPP等高标准经贸协定。
5. **有效防范化解重大经济金融风险**：防范化解优质头部房企风险，支持刚性和改善性住房需求。
6. **稳定粮食生产和推进乡村振兴**：粮食产量目标1.3万亿斤以上。
7. **推动发展方式绿色转型**：单位GDP能耗降低2%左右。
8. **保障基本民生和发展社会事业**：把促进青年特别是高校毕业生就业工作摆在更加突出的位置。

### 2.3 机构改革 —— 2023年两会最大看点

2023年两会期间，通过了根据党的二十届二中全会《党和国家机构改革方案》制定的国务院机构改革方案，涉及**13项改革举措**，聚焦六大领域：科学技术、金融监管、数据管理、乡村振兴、知识产权、老龄工作。

**核心改革措施**：

| 领域 | 改革内容 |
|------|---------|
| 科学技术 | 重新组建科学技术部，组建**中央科技委员会**，科技部强化宏观管理职能，不再参与具体项目评审 |
| 金融监管（重中之重，占6项） | 组建**国家金融监督管理总局**（统一负责除证券业外金融业监管）；证监会调整为国务院直属机构；央行分支机构撤销大区分行体制；深化地方金融监管体制改革 |
| 数据管理 | 组建**国家数据局**（国家发改委管理），统筹数字中国、数字经济、数字社会建设 |
| 乡村振兴 | 国家乡村振兴局职责划入农业农村部，加挂国家乡村振兴局牌子 |
| 老龄工作 | 应对人口老龄化职责由卫健委划入民政部 |
| 编制精减 | 中央国家机关各部门人员编制统一按**5%**比例精减，5年过渡期 |

**考点提示**：此次机构改革以**加强党中央集中统一领导为统领**，金融领域改革是重中之重，体现防范金融系统性风险、加强穿透式监管的鲜明导向。改革后除国务院办公厅外，国务院组成部门仍为**26个**。

## 三、2024年全国两会

> 2024年3月5日，国务院总理李强在第十四届全国人民代表大会第二次会议上作《政府工作报告》。
> 这是李强总理的首份政府工作报告。总基调：**稳中求进、以进促稳、先立后破**。

### 3.1 政府工作报告核心数据

| 指标 | 2024年目标 | 2024年实际 |
|------|-----------|-----------|
| GDP增速 | 5%左右 | **5.0%**（总量134.9万亿元） |
| CPI涨幅 | 3%左右 | **0.2%**（连续第二年远低于目标） |
| 城镇新增就业 | 1200万人以上 | **1256万人** |
| 城镇调查失业率 | 5.5%左右 | 平均**5.1%** |
| 居民收入增长 | 与经济增长同步 | 实际增长**5.1%** |
| 粮食产量 | 1.3万亿斤以上 | **1.4万亿斤**（首次跃上新台阶） |
| 赤字率 | 3%（赤字规模4.06万亿元） | — |
| 专项债 | 3.9万亿元 | — |
| 超长期特别国债 | 1万亿元（首次发行） | — |
| 单位GDP能耗 | 降低2.5%左右 | — |

### 3.2 重点政策部署 —— 多个"首次"标志性提法

**十大政府工作任务**（"新质生产力"首次列为第一项任务）：

1. **大力推进现代化产业体系建设，加快发展新质生产力**（列为首位，首次写入政府工作报告）
   - 培育新兴产业和未来产业：氢能、新材料、创新药、生物制造、商业航天、低空经济等
   - 开展**"人工智能+"行动**（首次提出）
   - 加快形成全国一体化算力体系
   - 开辟量子技术、生命科学等新赛道

2. **深入实施科教兴国战略**：强化基础研究系统布局，完善"揭榜挂帅"机制。

3. **着力扩大国内需求**：推动消费品以旧换新，培育智能家居、文娱旅游、体育赛事、国货"潮品"等新增长点。中央预算内投资安排7000亿元。

4. **坚定不移深化改革**：全面落实促进民营经济发展壮大的配套举措；加快全国统一大市场建设；谋划新一轮财税体制改革。

5. **扩大高水平对外开放**：**全面取消制造业领域外资准入限制措施**；打造"投资中国"品牌。

6. **统筹发展和安全**：优化房地产政策，对不同所有制房企合理融资一视同仁；稳妥处置地方债务、中小金融机构风险。

7. **抓好"三农"工作**：严守耕地红线，适当提高小麦最低收购价。

8. **推动城乡融合和区域协调发展**。

9. **加强生态文明建设**。

10. **切实保障和改善民生**：居民医保人均财政补助标准提高30元；城乡居民基础养老金月最低标准提高20元。

**五大金融"篇章"**首次在政府工作报告中完整提出：科技金融、绿色金融、普惠金融、养老金融、数字金融。

### 3.3 2024年关键亮点汇总

| 亮点 | 说明 |
|------|------|
| **"新质生产力"** | 首次写入政府工作报告，并列为第一项任务 |
| **"人工智能+"** | 首次提出，标志AI从技术概念上升为国家行动 |
| **超长期特别国债** | 首次连续多年发行机制，2024年首发1万亿元 |
| **"投资中国"品牌** | 新提法，配合全面取消制造业外资准入限制 |
| **消费品以旧换新** | 本轮大规模设备更新和消费品以旧换新的启动信号 |
| **宏观政策取向一致性** | 将非经济性政策纳入宏观政策取向一致性评估 |

## 四、2025年全国两会

> 2025年3月5日，国务院总理李强在第十四届全国人民代表大会第三次会议上作《政府工作报告》。
> 背景：**"十四五"收官之年**，要为"十五五"良好开局打牢基础。总基调：**稳中求进、以进促稳，守正创新、先立后破**。

### 4.1 政府工作报告核心数据

| 指标 | 2025年目标 | 2025年实际 |
|------|-----------|-----------|
| GDP增速 | 5%左右 | **5.0%**（总量140.19万亿元） |
| CPI涨幅 | 2%左右 | **0%（持平）**——连续三年物价低迷 |
| 城镇新增就业 | 1200万人以上 | **1267万人** |
| 城镇调查失业率 | 5.5%左右 | 平均**5.2%** |
| 居民收入增长 | 与经济增长同步 | 与经济增长**同步** |
| 粮食产量 | 1.4万亿斤左右 | **1.43万亿斤** |
| 赤字率 | 4%左右（赤字规模5.66万亿元） | — |
| 专项债 | 4.4万亿元 | — |
| 超长期特别国债 | 1.3万亿元 | — |
| 特别国债 | 5000亿元（补充国有大行资本） | — |
| 合计新增政府债务 | 11.86万亿元 | — |
| 单位GDP能耗 | 降低3%左右 | — |

### 4.2 十大重点任务

1. **大力提振消费、提高投资效益，全方位扩大国内需求**：安排超长期国债3000亿元支持以旧换新；落实和优化休假制度；扩大健康、养老、托幼、家政等多元化服务供给。

2. **因地制宜发展新质生产力**：培育生物制造、量子科技、具身智能、6G等未来产业；持续推进"人工智能+"行动；大力发展智能网联新能源汽车、智能机器人等。

3. **深入实施科教兴国战略**：扩大高中阶段教育学位供给，**逐步推行免费学前教育**；强化关键核心技术攻关和前沿性、颠覆性技术研发。

4. **推动标志性改革举措加快落地**：纵深推进全国统一大市场建设；综合整治**"内卷式"竞争**；开展规范涉企执法专项行动（坚决防止违规异地执法）。

5. **扩大高水平对外开放**：扩大电信、医疗、教育等领域开放试点；推动互联网、文化等领域有序开放。

6. **有效防范化解重点领域风险**：持续用力推动**房地产市场止跌回稳**；稳妥化解地方政府债务风险。

7. **着力抓好"三农"工作**：粮食产量目标1.4万亿斤左右；启动粮食产销区省际横向利益补偿；有序推进第二轮土地承包到期后再延长30年试点。

8. **推进新型城镇化和区域协调发展**：推动农业转移人口纳入住房保障体系；支持**经济大省挑大梁**。

9. **协同推进降碳减污扩绿增长**：建立一批零碳园区、零碳工厂；健全绿色消费激励机制。

10. **加大保障和改善民生力度**：城乡居民基础养老金最低标准再提高20元；居民医保人均财政补助标准再提高30元；**制定促进生育政策，发放育儿补贴**；加强灵活就业和新就业形态劳动者权益保障。

### 4.3 "十五五"规划编制启动

2025年将开展**"十五五"规划编制工作**，深入分析"十五五"时期新的阶段性特征，谋划重大战略任务、重大政策举措、重大工程项目。

**"十五五"时期五大类主要目标指标**：

| 类别 | 核心指标 |
|------|----------|
| 经济发展 | GDP增长保持在合理区间 |
| 创新驱动 | 全社会研发经费投入年均增长7%以上 |
| 民生福祉 | 就业、收入、教育、医疗、健康、"一老一小"等7项指标 |
| 绿色低碳 | 单位GDP二氧化碳排放累计降低17% |
| 安全保障 | 粮食、能源生产能力2项指标 |

**四大重大战略任务**：突出推动高质量发展；突出做强国内大循环；突出推进全体人民共同富裕；突出统筹发展和安全。

### 4.4 2025年重大政策转向信号

- **货币政策转向"适度宽松"**：这是自2011年以来的首次基调调整。2011-2024年连续14年表述为"稳健"，2025年改为"适度宽松"。
- **赤字率首次突破4%**：从2023/2024年的3%大幅上调至4%，赤字规模5.66万亿元。
- **"内卷式"竞争首次写入政府工作报告**：信号意味深长，指向部分行业产能过剩、恶性竞争问题。
- **育儿补贴首次在国家层面提出**：标志人口政策从"放开生育"转向"鼓励生育"。

## 五、2026年全国两会

| CPI涨幅 | 2%左右 |

| 城镇调查失业率 | 5.5%左右 |

### 5.2 十大任务详解

> 每条任务附考点链接。

**任务一：着力建设强大国内市场**
坚持内需主导，统筹促消费和扩投资。深入实施提振消费专项行动，充分挖掘释放有效投资潜力。安排超长期特别国债2500亿元支持消费品以旧换新，设立1000亿元财政金融协同促内需专项资金。8000亿元超长期国债用于"两重"建设（国家重大战略实施和重点领域安全能力建设），发行新型政策性金融工具8000亿元。
> **考点**：扩大内需战略、消费与投资的双轮驱动、财政政策工具创新

**任务二：加紧培育壮大新动能**
因地制宜发展新质生产力，建设现代化产业体系。打造**集成电路、航空航天、生物医药、低空经济**等新兴支柱产业；培育**未来能源、量子科技、具身智能、脑机接口、6G**等未来产业；深化拓展"人工智能+"，推进制造业"智改数转网联"。
> **考点**：新质生产力、现代化产业体系、"人工智能+"行动的深化、未来产业布局

**任务三：加快高水平科技自立自强**
加强原始创新和关键核心技术攻关，推动科技创新和产业创新深度融合，一体推进教育科技人才发展。建设北京、上海、粤港澳大湾区国际科技创新中心。
> **考点**：科技自立自强、教育科技人才一体化、国际科技创新中心建设

**任务四：持续深化重点领域改革**
纵深推进全国统一大市场建设，推进财税金融体制改革，充分激发各类经营主体活力。完善民营经济促进法配套法规政策。
> **考点**：全国统一大市场、民营经济促进法、财税体制改革、经营主体活力

**任务五：进一步扩大高水平对外开放**
积极扩大自主开放，推动外贸稳规模优结构，扩大双向投资合作，高质量共建"一带一路"。进一步扩大增值电信、生物技术、外商独资医院等领域开放试点。
> **考点**：高水平对外开放、自主开放、高质量共建"一带一路"、服务业开放

**任务六：扎实推进乡村全面振兴**
毫不放松抓好粮食生产，实施常态化精准帮扶，持续推进农村改革发展。**全面开展第二轮土地承包到期后再延长30年整省试点**。
> **考点**：粮食安全、防止返贫、土地承包延期、乡村全面振兴

**任务七：推动新型城镇化和区域协调发展**
深入推进以人为本的新型城镇化，推进农业转移人口市民化；不断增强区域发展协调性，推动西部大开发、东北振兴、中部崛起、东部现代化。
> **考点**：新型城镇化、农业转移人口市民化、区域协调发展战略

**任务八：更大力度保障和改善民生**
促进高质量充分就业，实施稳岗扩容提质行动；完善免费学前教育政策，增加普通高中学位供给；居民医保人均财政补助标准提高24元；城乡居民基础养老金月最低标准再提高20元；**加强初婚初育家庭住房保障**。
> **考点**：高质量就业、"一老一小"民生保障、初婚初育住房保障（人口政策新动向）

**任务九：加快推动全面绿色转型**
加强生态环境综合治理，大力发展绿色低碳经济，积极稳妥推进碳达峰碳中和。**设立国家低碳转型基金**，培育氢能、绿色燃料等新增长点。
> **考点**：绿色低碳转型、碳达峰碳中和、国家低碳转型基金（新增制度安排）

**任务十：加强重点领域风险防范化解和安全能力建设**
着力稳定房地产市场（**因城施策控增量、去库存、优供给**）；积极有序化解地方政府债务风险；积极稳妥化解金融领域风险；维护国家安全和社会稳定。
> **考点**：房地产新模式、地方债务化解、统筹发展与安全

### 6.3 趋势分析

**1. GDP目标：从"点目标"到"区间目标"**
2023-2025年GDP目标均为"5%左右"，2026年首次设定为"4.5%—5%"区间。这一变化反映：经济增长中枢在"十四五"末至"十五五"初出现结构性下移；同时区间目标给予宏观政策更大灵活性，体现务实态度。

**2. 赤字率：从"3%铁律"到"4%常态化"**
2023-2024年赤字率维持在3%，2025-2026年跃升至4%。这一突破指向积极财政政策实质性加力，超长期特别国债制度化（连续第三年1.3万亿级别），财政扩张已成新常态。

**3. 物价低迷是贯穿四年的最大挑战**
2023年CPI 0.2%、2024年0.2%、2025年0%——连续三年物价几乎零增长，供强需弱格局持续。2025年起CPI目标从3%下调至2%，2026年继续维持2%，政策诚意度提高但仍面临较大实现难度。2026年报告首次将**"物价合理回升"与"经济稳定增长"并列**，力度空前。

**4. 新质生产力：从概念到体系**
2023年：尚未出现→2024年：首次写入并列为第一任务→2025年："因地制宜"发展→2026年：具体化为"五大新兴支柱产业+五大未来产业"，路径越来越清晰。

**5. 人口与生育政策：三年三级跳**
2024年："一老一小"→2025年：首次提出"发放育儿补贴"→2026年：**"加强初婚初育家庭住房保障"**。政策从应对老龄化逐步转向更为积极的鼓励生育立场。

**6. 房地产：三年政策路径清晰可辨**
2023年"防范化解优质头部房企风险"→2024年"对不同所有制房企一视同仁"→2025年"止跌回稳"→2026年"控增量、去库存、优供给"九字方针。从被动防风险到主动调结构。

**7. 就业：持续超预期完成**
四年新增就业均超过目标——1244万→1256万→1267万，就业形势超出预期，是宏观经济中表现最稳定的指标之一。

### 7.4 面试模拟题

**模拟题一：综合分析类**

> 2024年政府工作报告首次将"新质生产力"列为首要任务，并提出开展"人工智能+"行动。2026年"十五五"规划进一步将"人工智能+"上升为"全面实施"。请谈谈你对这一政策演变的理解。

**答题要点**：
1. 定位：新质生产力是高质量发展的核心驱动力，是应对新一轮科技革命和产业变革的战略选择。
2. 演变逻辑：2024年"首次写入+列为首位"（破题）→2025年"因地制宜"（纠偏、防一哄而上）→2026年"五大产业+五大未来产业"（细化落地），体现政策从理念到执行的递进。
3. 辩证分析：既要看到发展新质生产力的紧迫性（国际竞争、产业升级），也要关注"因地制宜"背后的深意——防止盲目投资、重复建设，防止"内卷式竞争"。
4. 延伸：科技创新、教育科技人才一体化、数字经济与实体经济融合。
5. 总结升华：以新质生产力推动中国式现代化，实现从"量的积累"到"质的飞跃"。

**模拟题二：应急应变/组织计划类**

> 假如你是某市发改委工作人员，2026年政府工作报告提出"因城施策控增量、去库存、优供给"，同时要"加强初婚初育家庭住房保障"。领导要求你起草本市落实方案，你会从哪些方面入手？

**答题要点**：
1. **摸清底数**：调研本市商品房库存量、去化周期、常住人口住房需求（特别是初婚初育家庭数量及住房条件）。
2. **控增量**：合理控制新增住宅用地供应节奏，根据去化周期动态调整出让计划。
3. **去库存**：探索政府收储存量商品房用于保障性住房；鼓励"以旧换新"；优化限购限售政策。
4. **优供给**：增加中小户型、保障性住房供给；提升住房品质（绿色、智能、适老化）。
5. **初婚初育家庭专项**：设立购房补贴或贷款利率优惠；优先配售保障性住房；探索"共有产权"模式降低门槛。
6. **配套措施**：同步考虑教育、医疗、托幼等公共服务配套，让"住有所居"有更强的幸福感。
7. **防范风险**：注意不引发新的房地产泡沫，确保财政可持续。
