# 中央政治局会议（2023-2026·季度经济议题）

> 频次：每月（一般），季度经济议题集中于4月/7月/10月/12月
> 核心功能：年度经济工作的"过程管理"——年中定调+年底预热
> 考试重要度：★★★★ / 覆盖：申论 · 面试 · 行测常识
> 整理日期：2026-05-10

---

## 一、制度定位

- **一般规律**：中央政治局每月召开一次会议，其中**4月、7月、10月、12月**的会议通常包含季度经济形势分析。
- **4月政治局会议**：分析一季度经济形势，评估年初开局状况（"开局定全年"）。
- **7月政治局会议**：分析上半年经济形势，部署下半年经济工作（**"年中定调"关键节点**）。
- **12月政治局会议**：为中央经济工作会议"预热"，先定大方向。
- **特殊情形**：2024年9月政治局会议突然讨论经济（通常9月不谈经济），被视为"一揽子增量政策"启动的关键信号。

### 三会"流水线"关系

```
中央经济工作会议（每年12月，年度总定调+大方向）
        ↓
两会上《政府工作报告》（次年3月，法律转化+量化目标+具体措施）
        ↓
中央政治局季度会议（4月/7月/10月/12月，过程监控+微调纠偏）
        ↓
国务院常务会议（每月2-3次，日常执行落实）
```

- **中央经济工作会议**：党中央层面的战略决策——"明年经济工作怎么干"
- **两会政府工作报告**：将党的意志通过法定程序转化为国家目标——GDP增速、CPI、赤字率等量化指标
- **政治局季度会议**：监控执行进度，根据形势变化微调政策发力方向
- **国务院常务会议**：将决策部署转化为具体政策措施的日常高频平台（总理主持，固定参加仅10人）

## 二、2023年关键政治局会议

### 2.1 7月24日——部署下半年经济工作（年中定调）

- **形势判断**：国民经济持续恢复、总体回升向好，但"疫情防控平稳转段后，经济恢复是一个**波浪式发展、曲折式前进**的过程"（经典表述）
- **四大问题**：国内需求不足、一些企业经营困难、重点领域风险隐患较多、外部环境复杂严峻
- **关键部署**：
  - 精准有力实施宏观调控，加强**逆周期调节**和政策储备
  - 提振汽车、电子产品、家居等大宗消费，推动体育休闲、文化旅游等服务消费
  - **活跃资本市场，提振投资者信心**（首次在政治局会议提出）
  - 制定出台促进民间投资的政策措施
  - 有效防范化解地方债务风险，制定实施**一揽子化债方案**
  - 适时调整优化房地产政策

### 2.2 4月28日——分析一季度经济形势

- 一季度GDP增长4.5%，开局良好
- 三重压力得到缓解，但内生动力还不强

### 2.3 12月8日——定调2024年经济工作方向（预热经济工作会议）

- 为中央经济工作会议"预热"
- 提出"稳中求进、以进促稳、先立后破"
- 强调"增强宏观政策取向一致性"

## 三、2024年关键政治局会议

### 3.1 4月30日——分析一季度经济形势

- Q1 GDP增长5.3%，**"开局良好、回升向好"**
- **"坚持乘势而上，避免前紧后松"**
- 统筹研究消化存量房产和优化增量住房
- 及早发行并用好超长期特别国债，加快专项债发行使用进度
- 决定7月召开二十届三中全会

### 3.2 7月30日——部署下半年经济工作

- Q2 GDP增速回落至4.7%，下行压力加大
- **"宏观政策要持续用力、更加给力"**
- 以提振消费为重点扩大国内需求，多渠道增加居民收入

### 3.3 ★★★ 9月26日——超常规部署（年度最大政策转折点）

> **背景**：通常9月政治局会议不讨论经济议题。此次**破例开会**，被视为全年经济政策的重大转折，也是"一揽子增量政策"启动的关键信号。

- **形势判断**：**"正视困难、坚定信心，切实增强做好经济工作的责任感和紧迫感"**
- **六大政策信号**：
  1. **努力完成全年5%左右增长目标**（坚定目标不动摇）
  2. 加大财政货币逆周期调节：降准、降息、发行超长期特别国债和专项债
  3. **促进房地产市场止跌回稳**（政治局会议首次明确提出）
  4. **提振资本市场**：引导中长期资金入市
  5. 加大引资稳资力度
  6. 守住兜牢民生底线
- **效果**：Q4 GDP增长5.4%，全年最终实现5.0%

### 3.4 12月9日——定调2025年经济工作方向（预热经济工作会议）

- 实施**更加积极的财政政策**和**适度宽松的货币政策**（定调前移）
- "加强超常规逆周期调节"
- 稳住楼市股市

## 四、2025年关键政治局会议

### 4.1 4月25日——分析一季度经济形势

- Q1 GDP增长5.4%，超预期"开门红"
- 但4月美国对华大幅加征"对等关税"，外部冲击显著加大
- **首次提出"四稳"**：稳就业、稳企业、稳市场、稳预期（稳就业居首位）
- 重提**"超常规逆周期调节"**，要求既定政策早出台早见效，根据形势变化及时推出增量储备政策
- **"适时降准降息"**，创设新的结构性货币政策工具，设立新型政策性金融工具
- 提高中低收入群体收入，大力发展服务消费，设立服务消费与养老再贷款
- 打造新兴支柱产业，加快实施"人工智能+"行动
- 持续巩固房地产市场稳定态势
- **"持续稳定和活跃资本市场"**

### 4.2 7月30日——部署下半年经济工作

- 上半年GDP增长5.3%，**"经济运行稳中有进，高质量发展取得新成效"**
- 宏观政策**"持续发力、适时加力"**（新表述）
- 纵深推进全国统一大市场建设，**依法依规治理企业无序竞争**（首次在政治局会议出现"反内卷"提法）
- **增强国内资本市场的吸引力和包容性**（首次提出）
- 稳住外贸外资基本盘，帮助受冲击较大的外贸企业
- 决定10月召开二十届四中全会，研究关于制定"十五五"规划的建议

### 4.3 12月8日——定调2026年经济工作方向（预热经济工作会议）

- "十四五"即将圆满收官，全年经济社会发展主要目标将顺利实现
- **2026年基调**："坚持稳中求进、提质增效"
- 继续实施更加积极的财政政策和适度宽松的货币政策
- 发挥存量政策和增量政策集成效应，加大逆周期和跨周期调节力度
- 编制"十五五"规划及专项规划

## 五、2026年主要政治局会议（截至5月）

### 5.1 ★★★ 4月28日——"十五五"开局首次经济评估

- **形势判断**：经济起步有力，Q1 GDP增长5.0%，**主要指标好于预期**，彰显强大韧性和活力
- 持续稳中向好的**基础还需进一步巩固**
- **政策基调**：用好用足宏观政策，增强货币政策**前瞻性、灵活性、针对性**
- **重点部署**：
  - 扩大优质商品和服务供给，推动**消费升级**
  - 深入实施服务业扩能提质行动
  - ★★★ 加强**六张"网"**规划建设：水网、新型电网、算力网、新一代通信网、城市地下管网、物流网（**首次提出**）
  - 全面实施**"人工智能+"行动**，发展智能经济新形态
  - 纵深推进全国统一大市场建设，深入整治"内卷式"竞争
  - 努力稳定房地产市场，扎实推进城市更新
  - 有序化解地方政府债务风险
  - **稳定和增强资本市场信心**
  - 强化就业优先政策导向

## 六、政治局会议与经济工作会议的关系

### 6.1 "预热+纠偏"双重角色

```
政治局12月会议 → 预热 → 中央经济工作会议（定大方向）
        ↓
政治局4月会议 → 开局评估 → 一季度GDP+政策微调
        ↓
政治局7月会议 → 年中定调 → 研判下半年经济走势
        ↓
政治局9月会议（特殊情形）→ 紧急纠偏 → 应对突发风险
        ↓
政治局12月会议 → 预热 → 新一轮经济工作会议
```

- **"预热"**：每年12月政治局会议先行讨论经济工作，为随后召开的中央经济工作会议定好大方向
- **"纠偏"**：季度会议（特别是7月和破例9月）根据经济数据及时调整政策节奏和力度
- **"定调前移"**：2024年12月政治局会议首次将财政"更加积极"、货币"适度宽松"这两个重大政策转向在政治局会议层面先行释放，中央经济工作会议随后详细部署

### 6.2 关键时间节点与政策信号对照

| 节点 | 政治局会议信号 | 后续经济工作会议 |
|------|---------------|------------------|
| 2023年12月 | "稳中求进、以进促稳、先立后破" | 12字方针正式确立，新质生产力首次写入 |
| 2024年9月（破例） | 一揽子增量政策启动 | —（年内纠偏） |
| 2024年12月 | "更加积极"+"适度宽松"定调前移 | 24字方针确立，货币财政双扩张 |
| 2025年12月 | "稳中求进、提质增效"，十五五预热 | 八个坚持，十五五开局部署 |
| 2026年4月（最新） | "六网建设"首次提出 | —（待观察） |

## 七、考试视角

### 7.1 三会"流水线"关系（申论写作框架）

```
中央经济工作会议（每年12月）
  └─ 定大方向：总基调 + 政策取向 + 重点任务
        ↓
两会《政府工作报告》（次年3月）
  └─ 量化落地：GDP目标、CPI目标、赤字率、专项债额度等
        ↓
中央政治局季度会议（4月/7月/10月/12月）
  └─ 过程监控：根据数据调整政策力度和节奏
        ↓
国务院常务会议（每月2-3次）
  └─ 日常执行：具体政策措施出台
```

**申论应用**：写"政策部署"类文章时，可用此框架说明政策从"定调"到"落地"到"纠偏"的逻辑链条。

### 7.2 申论价值

- **政治局会议体现"过程管理"思维**：经济政策不是一步到位，而是根据数据不断调整——这种"动态治理"是中国宏观调控的鲜明特色
- **"年中定调"（7月）是政策转向的敏感窗口**：如2024年7月提出"宏观政策要持续用力、更加给力"，随后9月破例加码
- **"预热"机制**：12月政治局会议先定大方向，为经济工作会议铺路，体现出"党中央集中统一领导"的决策流程
- **"反内卷"成为高频词**：从2024年经济工作会议到2025年7月政治局会议再到2026年4月，整治"内卷式"竞争连续出现在顶层设计中，是申论面试的热点话题

### 7.3 行测常识考点

| 考点 | 内容 |
|------|------|
| 政治局会议讨论经济的常规月份 | 4月、7月、10月、12月（7月最关键："年中定调"） |
| 2024年9月政治局会议的特殊性 | 破例讨论经济，启动一揽子增量政策 |
| "四稳" | 稳就业、稳企业、稳市场、稳预期（2025年4月首次提出） |
| "反内卷"在政治局会议首次出现 | 2025年7月政治局会议（"依法依规治理企业无序竞争"） |
| 六张"网" | 水网、新型电网、算力网、新一代通信网、城市地下管网、物流网（2026年4月政治局会议首次提出） |
| 政治局12月会议的"预热"功能 | 先于中央经济工作会议定大方向 |
| 货币政策"适度宽松" | 2024年12月政治局会议定调前移 |
| "超常规逆周期调节" | 2024年12月政治局会议首提，2025年4月重提 |
| "人工智能+"行动 | 2025年4月政治局会议强调，2026年4月全面实施 |
| "十五五"规划编制 | 2025年10月二十届四中全会研究建议，2025年12月政治局会议要求编制 |

### 7.4 面试模拟题

#### 题一：综合分析类

> 2024年9月26日，中央政治局破例召开会议讨论经济形势，部署一揽子增量政策。你认为这次"破例"会议释放了什么信号？

**答题要点**：
1. **背景分析**：通常9月不谈经济，破例开会说明经济下行压力超预期——Q3 GDP增速仅4.6%
2. **信号一：目标坚定不动摇**——"努力完成全年5%左右增长目标"
3. **信号二：政策果断转向**——从观望到果断出手，财政货币双双加力
4. **信号三：房地产和资本市场两大抓手**——"止跌回稳"+"提振资本市场"
5. **启示**：政策决策不是僵化的，而是动态响应的——这体现了"实事求是"的工作方法

#### 题二：计划组织类

> 2026年4月政治局会议提出加强"六网建设"（水网、新型电网、算力网、新一代通信网、城市地下管网、物流网）。你所在城市准备制定"六网建设"三年行动计划，领导让你牵头调研，你会如何组织？

**答题要点**：
1. **调研目的**：摸清本市六网现状、短板和建设需求
2. **调研对象**：水务局、电力公司、经信局（算力/通信）、住建局（管网）、交通局/商务局（物流）
3. **调研重点**：现有设施承载能力、老旧管网改造需求、算力和通信基础设施与发达地区差距、物流节点布局合理性
4. **方式方法**：实地勘察+专家座谈+数据收集+先进城市对标
5. **成果输出**：各网现状评估报告+优先项目清单+三年投资估算+配套政策建议
