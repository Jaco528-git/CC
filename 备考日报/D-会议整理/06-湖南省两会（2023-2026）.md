# 湖南省两会（2023-2026）

> 时间：每年1月 | 地点：长沙
> 与全国两会关系：地方先开（1月），全国后开（3月），地方为全国"预热"
> 考试重要度：★★★★★ / 覆盖：申论 · 面试 · 行测常识 / 选调必知
> 整理日期：2026-05-10

---

## 一、制度定位：省两会与全国两会

### 1.1 时间与内容关系

- **时间顺序**：湖南省两会一般在每年**1月**召开，全国两会在**3月**召开。地方先行为全国"预热"，地方报告中的政策诉求在一定程度上反映全国政策走向。
- **内容关系**：湖南省政府工作报告中的GDP目标通常高于全国目标（体现地方"跳起来摘桃子"），实际增速往往与全国走势高度同步。
- **代表联动**：湖南省选出的全国人大代表（约110余名）参加全国两会，将湖南诉求带入国家议事平台。

### 1.2 湖南在全国政治版图中的角色

- **中部崛起重要支点**：湖南是中部六省之一，2025年GDP达5.53万亿，稳居全国前十强。
- **"一带一部"定位**：东部沿海地区和中西部地区过渡带、长江开放经济带和沿海开放经济带结合部（习近平总书记2013年提出）。
- **经济大省担当**：2025年政府工作报告自评"扛牢了经济大省挑大梁的政治责任"。

## 五、2026年湖南省政府工作报告

> 湖南省第十四届人民代表大会第四次会议，2026年2月3日开幕，省长毛伟明作政府工作报告。

- 五年GDP增量超**1.36万亿元**，人均GDP达**1.19万美元**
- 2025年GDP达5.53万亿元，稳居全国前十强
- "三高四新"美好蓝图渐成生动现实

### 5.2 2026年主要预期目标

| 指标 | 目标 |
|------|------|
| 地区生产总值增长 | **5%左右**（争取更好结果） |
| 规模工业增加值增长 | 6% |
| 社会消费品零售总额增长 | 4.5%以上 |
| 固定资产投资 | 实现正增长 |
| 进出口总额增长 | 3% |
| 地方一般公共预算收入增长 | 1.5%左右 |
| 城镇新增就业 | 70万人 |
| 居民收入增速 | 高于经济增速 |
| CPI涨幅 | 2%左右 |
| 粮食产量 | 615亿斤左右 |

### 5.3 十个方面重点工作

1. **全力推动经济稳定增长** — 释放消费潜力，扩大有效投资（389个省重点项目，总投资2万亿元）
2. **加快建设现代化产业体系** — 壮大"4×4"现代化产业体系，推进"人工智能+"行动
3. **深入推进创新驱动发展** — 推进"1+2"国家实验室体系建设，全社会研发投入强度提升至2.62%
4. **全面深化改革扩大开放** — 推进零基预算改革，长沙获批中欧班列国家集结中心
5. **强化区域协调联动发展** — 长株潭全域纳入国家要素市场化配置综合改革试点
6. **扎实促进乡村全面振兴** — "和美湘村"建设
7. **着力加强生态文明建设** — 森林覆盖率保持在54%以上
8. **持续发展社会民生事业** — 十大重点民生实事601亿元
9. **更好担负新时代文化使命** — 文化和科技、文化和旅游"两个融合"
10. **牢牢守住安全稳定底线** — 安全生产治本攻坚

### 5.4 十大重点民生实事（601亿元）

总投入约**601亿元**，比2025年增加30亿元，共23个具体项目（新办11个、续办12个）：

- **教育**：扩充优质公办高中学位8万个；提质升级1000所中小学食堂；新增省属高校校舍30万平方米
- **就业创业**：城镇新增就业70万人；"技兴三湘一人一技"职业技能培训100万人次；发放创业担保贷款62亿元
- **养老**：建设2000张"时光守护"床位；建设25个老年认知障碍照护友好社区
- **便民服务**：新建1000家"爱心驿站"；流动供销服务乡村网点不少于1600个；新建农村4G/5G基站4700个

### 5.5 重大基础设施

- **6条高铁**：邵永、铜吉、长赣、宜常、永清广、黔吉
- **17条高速公路**：京港澳和沪昆扩容、张官等
- **水利**：梅山灌区、金塘冲水库等
- **能源**：推进"源网荷储"一体工程

### 6.2 GDP目标与实际增速差距分析

| 年份 | 目标增速 | 实际增速 | 差距 | 分析 |
|------|:------:|:------:|:------:|------|
| 2023 | 6.5% | 4.6% | **-1.9pp** | 差距最大年份。疫后复苏不及预期，外部需求走弱、房地产深度调整拖累 |
| 2024 | 6% | 4.8% | **-1.2pp** | 差距收窄但仍较显著。投资增速回落，消费恢复偏慢 |
| 2025 | 5.5% | 4.8% | **-0.7pp** | 差距进一步收窄。Q4高于全国0.2个百分点，但全年仍低于目标 |
| 2026 | 5% | — | — | 目标首次调至全国持平水平，信号转趋务实 |

> **关键趋势**：①目标逐年下调（6.5%→6%→5.5%→5%），反映对经济下行压力的客观认识；②差距逐年收窄（1.9→1.2→0.7pp），说明目标制定渐趋务实；③2026年首次将目标与全国对齐（5%左右），同时强调"争取更好结果"。

### 7.2 中央→湖南政策传导实例（政府工作报告视角）

以**2025年"两重""两新"政策**为例：

```
中央经济工作会议（2024.12）
  └→ 提出"两重"（国家重大战略实施和重点领域安全能力建设）
      "两新"（大规模设备更新和消费品以旧换新）
        └→ 湖南省政府工作报告（2025.1）
              └→ 量化落实：289个省重点项目、消费品以旧换新带动销售905亿元
                  └→ 十大技术攻关项目对接"两重"方向
```

这一链条体现了**"中央定调→地方政府工作报告量化分解→条块执行"**的标准化路径，是理解央地关系的关键框架。
