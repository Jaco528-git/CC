全会传统议题权力环节本届实况

三中全会重大经济改革定方向2024.7 300+项改革
四中全会党建或法治专题推改革2025.10 十五五规划建议★

口诀：一中选领导 / 二中议人事 / 三中推改革 / 四中本届定规划 / 五中待定

全国两会

全国人民代表大会 + 中国人民政治协商会议 / 每年3月

人大 vs 政协

  • 人大：最高国家权力机关，审议表决有法律效力
  • 政协：统一战线组织，讨论建议不表决
  • 人大提议案，政协提提案
  • 二者是选举民主+协商民主的结合

四年关键指标

  • 2023：GDP 5%左右 → 实际5.2%
  • 2024：人工智能+首次写入政府工作报告
  • 2025：赤字率4%，超长期特别国债1.3万亿
  • 2026：GDP首次区间目标 4.5%-5%

三会递进：党代会 定方向 → 中央全会 作部署 → 两会 法定化

中央经济工作会议

每年12月 / 定调次年经济总基调

年份总基调关键政策信号
2022稳字当头、稳中求进三稳：稳增长/稳就业/稳物价 / 六个更好统筹
2023稳中求进、以进促稳、先立后破新质生产力首次提出
2024稳中求进、以进促稳货币政策适度宽松14年首次 / 更加积极财政
2025稳中求进、提质增效十五五开局之年

流水线：经济工作会议（12月总定调）→ 两会报告（3月量化）→ 政治局季度会（过程监控）

政治局季度会议

4月/7月/10月/12月 / 经济工作的过程管理

常规节奏

  • 4月：分析一季度，开局定全年
  • 7月：年中定调，下半年部署
  • 10月：三季度形势研判
  • 12月：为经济工作会议预热

★ 2024.9 破例

  • 9月破例讨论经济 — 最大政策转折点
  • 加力推出增量政策
  • → 12月经济工作会14年首次适度宽松
  • 2026新提法：六网建设

湖南省级会议

中央决策的省级转化：国标到省标

湖南省两会（每年1月）

  • 时间在中央之前：地方1月，全国3月
  • GDP目标通常高于全国（跳起来摘桃子）
  • 2025年GDP 5.53万亿，全国前十
  • 一带一部定位

省委全会 + 省委经济工作会

  • 省委全会：每年至少2次，审议重大部署
  • 省委经济工作会：12月，对标中央
  • 政策传导差：中央到湖南 3-6个月
  • 案例：零基预算改革传导全过程
