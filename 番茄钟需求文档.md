# 番茄钟应用 — 需求文档

## 1. 项目概述
一个简洁、美观的番茄钟（Pomodoro Timer）浏览器应用，帮助用户使用番茄工作法进行时间管理。

## 2. 功能需求

### 2.1 计时模式
| 模式 | 默认时长 | 说明 |
|------|----------|------|
| 专注 | 25 分钟 | 倒计时工作阶段 |
| 短休息 | 5 分钟 | 两次专注之间的短暂休息 |
| 长休息 | 15 分钟 | 完成 4 个番茄后的一次较长休息 |

### 2.2 核心交互
- **开始/暂停**：点击按钮启动或暂停倒计时
- **重置**：重置当前模式倒计时至默认时长
- **任务计数**：自动统计已完成的番茄数量，每完成 4 个番茄自动推荐长休息
- **模式切换**：用户可在专注、短休息、长休息之间手动切换

### 2.3 通知
- 倒计时结束时发出**声音提示**
- 倒计时结束弹出**浏览器桌面通知**（需用户授权）
- 倒计时结束时页面标题闪烁提示

### 2.4 界面要求
- 居中显示当前倒计时（**MM:SS 格式**）
- 显示当前模式名称（专注 / 短休息 / 长休息）
- 显示已完成番茄数
- 控制按钮（开始/暂停、重置）
- 模式切换标签页
- 简洁、现代化 UI，柔和配色
- 响应式设计，适配桌面与移动端

### 2.5 数据持久化
- 使用 localStorage 保存：
  - 已完成番茄数
  - 当前模式设置
- 页面刷新后恢复状态

## 3. 技术方案
- 纯前端实现：**HTML + CSS + JavaScript**
- 不依赖任何第三方库或框架
- 单文件应用（所有代码位于一个 HTML 文件中）

## 4. 非功能性需求
- 倒计时精度 ≤ 1 秒
- 页面不重载，所有交互为客户端即时响应
- 符合 Web 无障碍基础要求（语义化标签、焦点状态）
